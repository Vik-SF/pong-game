# Pong Game

A simple, beautiful Pong game with 1-player and 2-player modes.

## How to Play

1. Open `index.html` in your web browser
2. Choose your game mode:
   - **1 Player**: Play against the AI
   - **2 Players**: Play with a friend

## Controls

- **Player 1**: 
  - `W` - Move paddle up
  - `S` - Move paddle down

- **Player 2** (in 2-player mode):
  - `↑` (Up Arrow) - Move paddle up
  - `↓` (Down Arrow) - Move paddle down

## Features

- Beautiful gradient UI design
- Smooth animations and transitions
- AI opponent with realistic difficulty
- Ball speed increases with each paddle hit
- Score tracking
- Easy navigation between menu and game

## Game Rules

- The ball bounces off the top and bottom walls
- Players must prevent the ball from passing their paddle
- Each time the ball passes a paddle, the opponent scores a point
- The game continues until you return to the menu

Enjoy the game!
