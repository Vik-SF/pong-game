## Troubleshooting

### Touch controls not showing?
- Refresh the page
- Make sure you're using a modern browser
- Try Safari on iOS or Chrome on Android

### Controls feel unresponsive?
- Close background apps
- Check if device is in low-power mode
- Ensure screen is clean and responding well

### Can't reach buttons?
- **Use drag control instead** - much easier!
- Just touch and drag anywhere on your side
- No need to reach specific buttons

### Game too small on phone?
- Rotate to landscape mode
- Add to home screen for fullscreen
- Zoom in before adding to home screen

### Online multiplayer laggy?
- Use WiFi, not cellular data
- Move closer to WiFi router
- Ensure stable connection
- Some lag is normal - game is still playable!
